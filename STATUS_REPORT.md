## 🚀 READY-TO-USE WORKFLOWS

### Workflow 1: Document-Based Exam
```
1. Register/Login
2. Upload document(s)
3. Go to Exam Generator
4. Select uploaded documents
5. Configure exam parameters
6. Generate exam from document context
7. View and manage results
```

### Workflow 2: Internet-Based Exam
```
1. Register/Login
2. Go to Exam Generator (no document needed)
3. Select "Internet" or "Both" as source
4. Topic will be scraped from Google/Wikipedia/ArXiv
5. Generate exam from scraped content
6. View results
```

### Workflow 3: Combined Approach
```
1. Register/Login
2. Upload some documents
3. Select "Both" source preference
4. Generate exam using:
   - Uploaded documents (high priority)
   - Internet sources (supplementary)
5. View comprehensive exam
```
